# SwiftSimpler
 
This app allows to play real time anf create your own beat.
Sound effects applied separately for each track and can be reordered by drag and drop.

Effects available:
- Reverb
- Delay
- Flanger 
- Distortion
- Two band equalizer 
- Hi & Low cutoff
